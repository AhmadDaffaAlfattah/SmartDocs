## ðŸ” SQL QUERIES YANG TERSEDIA

Di `FOLDER_SETUP.sql` sudah ada query-query berguna:

```sql
-- Lihat semua folder dengan parent
SELECT * FROM folders 
LEFT JOIN folders p ON f.parent_id = p.id;

-- Lihat root folder saja
SELECT * FROM folders WHERE parent_id IS NULL;

-- Lihat subfolder dari folder tertentu
SELECT * FROM folders WHERE parent_id = 1;

-- Get full path dari suatu folder
WITH RECURSIVE folder_path AS (...)
SELECT path FROM folder_path;

-- Hitung subfolder per folder
SELECT f.id, f.nama_folder, COUNT(c.id) as jumlah_subfolder
FROM folders f
LEFT JOIN folders c ON c.parent_id = f.id
GROUP BY f.id;
```

### Model - Parent-Child Relationship
```php
$folder = Folder::find(1);
$folder->parent();      // Get parent folder
$folder->children();    // Get all subfolder
$folder->getBreadcrumb();  // Get path array
$folder->getHierarchy();   // Get tree structure
```

### Controller - CRUD Operations
```php
public function index()           // List root folders
public function create()          // Show create form
public function store($request)   // Save new folder
public function show($id)         // View detail
public function edit($id)         // Show edit form
public function update()          // Update folder
public function destroy($id)      // Delete folder
public function tree()            // Return JSON tree
```

### Database - Self-Referencing FK
```sql
CREATE TABLE folders (
    id BIGINT PRIMARY KEY,
    nama_folder VARCHAR(255),
    parent_id BIGINT,  -- FK to self!
    FOREIGN KEY (parent_id) REFERENCES folders(id) ON DELETE CASCADE
);
```
